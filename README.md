# Estudos-Databricks-CSV-para-Tabela  

Sobre o projeto
O objetivo é demonstrar como transformar um arquivo CSV bruto em uma tabela estruturada e consultável dentro do ambiente Databricks, passando por todas as etapas do processo: exploração do armazenamento, leitura do arquivo, criação de tabela física e organização por schema.

O que foi feito:
Navegação pela estrutura de diretórios do DBFS
Leitura de arquivo CSV com inferência automática de schema usando PySpark
Exibição dos dados em formato tabular com display()
Persistência como tabela física em formato Delta Lake
Consulta dos dados via SQL
Criação e organização de schemas no Databricks
Limpeza do ambiente ao final do estudo
